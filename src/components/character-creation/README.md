"# Character Creation Wizard - Component Structure\n\nThe character creation wizard has been refactored into smaller, more manageable components.\n\n## File Structure\n\n### Core Files\n- `constants.js` - Shared constants (ABILITY_NAMES, ALIGNMENTS, etc.)\n- `utils.js` - Utility functions for validation and calculations\n- `character-creation-wizard-new.jsx` - Main orchestrator component\n\n### Step Components (One per wizard step)\n- `wizard-step-rules.jsx` - Ruleset selection (Step 1)\n- `wizard-step-basic.jsx` - Basic information (Step 2)\n- `wizard-step-race-class.jsx` - Race & Class selection (Step 3)\n- `wizard-step-abilities.jsx` - Ability scores (Step 4)\n- `wizard-step-skills.jsx` - Skills & Languages (Step 5)\n- `wizard-step-inventory.jsx` - Inventory management (Step 6)\n- `wizard-step-special.jsx` - Special features (Step 7)\n\n### Reusable UI Components\n- `wizard-header.jsx` - Wizard header with title and close button\n- `wizard-progress-bar.jsx` - Progress indicator\n- `wizard-footer.jsx` - Navigation buttons (Previous/Next/Submit)\n\n## Usage\n\n### Import the main wizard:\n```javascript\nimport { CharacterCreationWizard } from './character-creation';\n\n// Or import individual components:\nimport { WizardStepBasic, WizardStepAbilities } from './character-creation';\n```\n\n### Usage Example:\n```javascript\n<CharacterCreationWizard\n  onComplete={handleComplete}\n  onCancel={handleCancel}\n  allRaces={races}\n  allClasses={classes}\n  allSpells={spells}\n  allSpells2024={spells2024}\n/>\n```\n\n## Benefits of This Refactoring\n\n1. **Better Maintainability** - Each component has a single responsibility\n2. **Easier Testing** - Smaller components are easier to unit test\n3. **Reusability** - Individual step components can be used elsewhere\n4. **Clearer Structure** - Each file has a well-defined purpose\n5. **Simpler Code Review** - Changes are isolated to specific components\n\n## Migration Notes\n\n- The original `character-creation-wizard.jsx` has been preserved as a backup\n- The new component is exported as `CharacterCreationWizard` from the index\n- All existing functionality has been maintained\n- CSS classes remain compatible with the existing stylesheets\n",
